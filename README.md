# This project builds a machine-learning model to predict whether a person has diabetes based on medical features such as glucose level, BMI, age, etc.

# Steps Included
1. Load Dataset: Import and preview the dataset.
2. Data Preprocessing: Handle missing values, split features/labels, scale data.
3.Model Training: Logistic Regression model (easy to replace with Tree/Forest).
4.Evaluation: Accuracy, Confusion Matrix, Classification Report.
